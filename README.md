PyMeshUp
========

PyMeshUp aims to provide an easy way to make simple volume meshes using a script.

This is done by wrapping some of the features of pymeshlab and adding an gui.

To run the gui:

`python -m pymeshup.GUI`

Installation
-------------

`pip install pymeshup`

--------------------
